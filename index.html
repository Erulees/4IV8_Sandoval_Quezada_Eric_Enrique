Objetivo General

Desarrollar y estructurar una API RESTful utilizando Node.js y Express, implementando el patrón de diseño MVC mediante el uso de Express Router. El sistema es capaz de gestionar operaciones CRUD para entidades independientes y manejar relaciones entre tablas garantizando la integridad de los datos.

Lo que se aprendió

  • Modularidad con Express Router: Fragmentar la aplicación en rutas separadas para mayor organización.
  • Consultas Relacionales: Uso de INNER JOIN para combinar información de múltiples tablas en una sola respuesta.
  • Manejo de Middlewares: Configuración de intermediarios para procesar JSON y manejar errores.
  • Integridad Referencial: Aplicación de llaves foráneas para evitar registros huérfanos o inconsistentes.
